# How to clone
  * git repository: git@github.com:byoungokim/ATM.git
  * clone with the following
    ```bash
    git clone git@github.com:byoungokim/ATM.git
    ```

# How to run test
  * run following commands from the project root
  ```bash
  mkdir build
  cd build
  cmake ..
  make
  make test
  ```
